About Linux Software Engineer Position

We are looking for a talented Linux Software Engineer to join our product development team. As a Linux Software Engineer, you will be involved in developing, debugging, and deploying Linux-based embedded systems.

Key Responsibilities:

  • Develop, debug, and deploy Linux-based embedded systems.
  • Proficient in C and C++ programming languages.
  • Experience with Assembly for ARM and Intel processors.
  • Excellent understanding of Linux internals and Linux drivers.
  • Knowledge of I2C, USB, Ethernet, Wifi, and UEFI.
  • Understanding of boot code and hardware schematics.
  • Knowledge of CPU architecture including clock speed and cores.
  • Expertise in networking concepts including TCP, UDP, TCP/IP stack, switching, firewall, and VPN.
  • Experience with virtualization technologies such as KVM.

Required Skills

  • Deep understanding of the Linux operating system, including system calls, kernel configuration, and driver development.
  • Strong programming skills in C and C++.
  • Experience with scripting languages such as Python.
  • Experience with web API development, including RESTful API design and implementation.
  • Experience with Makefiles for building and deploying software on Linux systems.
  • Experience with ISO image creation and configuring installation environments.
  • Strong communication skills for collaboration with other developers and interaction with clients and stakeholders.
  • Excellent problem-solving skills for quick and effective issue resolution.
